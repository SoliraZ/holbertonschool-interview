## N Queens

This project solves the N queens puzzle using backtracking.

### File

- `0-nqueens.py`

### Usage

```bash
./0-nqueens.py N
```

### Rules

- If the number of arguments is wrong, prints: `Usage: nqueens N`
- If `N` is not a number, prints: `N must be a number`
- If `N < 4`, prints: `N must be at least 4`

### Output

- Prints one valid solution per line.
- Each solution is formatted as a list of `[row, col]` pairs.
