# Infinite Multiplication

This project implements a C program that multiplies two positive numbers of arbitrary length passed as command-line arguments.

## Files

- `0-mul.c`
- `holberton.h`

## Usage

```bash
./mul num1 num2
```

## Behavior

- Prints the multiplication result followed by a new line.
- If arguments are invalid (wrong count or non-digit characters), prints `Error` and exits with status `98`.
