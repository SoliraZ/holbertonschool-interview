# Rotate 2D Matrix

This project implements a function that rotates an `n x n` 2D matrix 90 degrees clockwise in-place.

## Prototype

```python
def rotate_2d_matrix(matrix):
```

## Behavior

- Edits the matrix in-place.
- Does not return anything.
- Assumes the matrix is square and not empty.

## Example

Input:

```python
[[1, 2, 3],
 [4, 5, 6],
 [7, 8, 9]]
```

Output after rotation:

```python
[[7, 4, 1],
 [8, 5, 2],
 [9, 6, 3]]
```
